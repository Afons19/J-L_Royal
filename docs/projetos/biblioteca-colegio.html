Sobre o Projeto

Um espaço de conhecimento e inspiração

A Biblioteca do Colégio Politécnico Celestino Sambambi foi concebida para ser muito mais do que um simples depósito de livros. Este projeto arquitetónico inovador cria um ambiente que inspira a aprendizagem, promove a colaboração e oferece diversos espaços para estudo e reflexão.

Com uma área total de 1.200 m², a biblioteca foi projetada para acomodar mais de 15.000 volumes, além de oferecer zonas de estudo individual, salas de grupo, áreas multimédia e um auditório para eventos académicos.

O design incorpora elementos da arquitetura contemporânea angolana, com claraboias que maximizam a iluminação natural e um sistema de ventilação passiva que reduz o consumo energético. Os materiais escolhidos priorizam a durabilidade e o baixo mantenimento, adequando-se ao clima local.
